## 1. 학습 콘텐츠

### 1.1 일상 영어 회화 (6개 주제)

| 주제 | 경로 | 핵심 내용 |
|------|------|-----------|
| 인사 & 소개 | `/conversation/greetings` | 기본 인사, 자기소개 패턴, 실전 대화 3개, 연습 7문제 |
| 일상생활 표현 | `/conversation/daily-life` | 아침 루틴, 날씨, 약속 잡기, 가정생활, 도움 요청 |
| 쇼핑 & 주문 | `/conversation/shopping` | 매장 쇼핑, 결제, 온라인 쇼핑, 반품/교환 |
| 여행 영어 | `/conversation/travel` | 공항, 입국심사, 호텔, 길 찾기, 응급상황 |
| 음식점 영어 | `/conversation/restaurant` | 예약, 주문, 특별 요청, 불만, 계산 |
| 전화 영어 | `/conversation/phone` | 전화 받기/걸기, 메시지, 비즈니스 전화, 마무리 |

**각 페이지 공통 구성:**
- 핵심 표현 15-20개 (영어 + 한국어 번역)
- 실전 대화 3-5개 (4-10 교환)
- 핵심 표현 정리 테이블
- 연습 문제 6-7개 (답안 토글)
- 문화 팁 박스
- 이전/다음 레슨 네비게이션

## 2. AI 챗봇

### 경로: `/ai-chat`

**기능:**
- OpenAI API (GPT-3.5/4) 기반 영어 대화 연습
- 6개 시나리오 선택 (자유 대화, 카페, 호텔, 면접, 레스토랑, 여행)
- 실시간 채팅 인터페이스 (채팅 버블 UI)
- 타이핑 인디케이터 (3개 점 애니메이션)
- 문법 교정 + 영어 답변
- Enter 키로 전송

**API 키 관리:**
- 사용자가 직접 OpenAI API 키 등록
- localStorage에 저장 (서버 미전송)
- 키 유효성 테스트 기능
- `sk-` 접두사 검증
- 마스킹 표시 (sk-...xxxx)

## 3. 음성 인식

### 경로: `/speech`

**기능:**
- Web Speech API (SpeechRecognition) 사용 (무료, API 키 불필요)
- 난이도별 예문 (초급/중급/고급)
- 마이크 버튼으로 녹음 시작/중지
- 실시간 음성 → 텍스트 변환
- 단어별 정확도 비교 (맞음: 초록, 틀림: 빨강)
- 전체 정확도 % 표시
- 녹음 중 펄스 애니메이션

**지원 브라우저:**
- Chrome (권장)
- Edge
- Safari (부분 지원)
- Firefox (미지원 → 안내 메시지 표시)

## 5. UI/UX 기능

### 테마
- 다크 모드 (기본): 딥 블루 (#0A1628)
- 라이트 모드: 밝은 배경 (#F8FAFC)
- 자동 모드: 시스템 설정 따름
- 토글 버튼 (Navbar)

### 다국어
- 한국어 (기본)
- English
- `t(ko, en)` 헬퍼 함수로 전환
- 토글 버튼 (Navbar)

### 반응형 디자인
- 데스크톱: 전체 네비게이션, 다중 컬럼 그리드
- 태블릿 (1024px): 2컬럼, 햄버거 메뉴
- 모바일 (768px): 단일 컬럼, 슬라이드인 메뉴
- 소형 모바일 (480px): 압축 레이아웃

### 애니메이션
- AOS (Animate On Scroll): fade-up, fade-down, fade-left, fade-right, zoom-in
- Glassmorphism 네비게이션 (backdrop-filter: blur)
- 히어로 파티클 효과
- 카드 호버 효과
- 단어 카드 3D 플립
- 마이크 버튼 펄스
- 타이핑 인디케이터

### 접근성
- 시맨틱 HTML (nav, main, section, article)
- ARIA 속성 (role, aria-label)
- 키보드 네비게이션
- `prefers-reduced-motion` 지원
- 포커스 트랩 (모달)
- 스킵 네비게이션
